hubplanner2triangu
----------

Description:
=============

Get .csv files from `export` folder and create `.xlsx` file.

Usage:
=================

### Options:

`-month` - for what period prepare report. Possible formats `2018-12`, `12`.

### Testing Notes

#### Integrational test

cli to run for current test data  
`go test -args -month 2018-12`

in releases tab.
